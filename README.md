# karaoke-bot
Бот для управления общим youtube плейлистом
# Инструкция по деплою (Linux)
На машину загрузить /deploy/karaoke_bot.jar

В директории создать файл .env в котором должны быть указаны переменные
BOT_TOKEN='токен телеграм'
BOT_USERNAME='юзернейм телеграм'
YT_PLAYLIST_ID='айди плейлиста на ютубе'
YT_OAUTH_METHOD=device // метод аутентификации
YT_TOKENS_DIR=token // папка в которую загрузится токен

Установить jre 25 версии

Запустить java -jar karaoke_bot.jar

Следовать указаниям консоли

Если необходимо, запустить процесс systemd который поддерживает работоспособность процесса
